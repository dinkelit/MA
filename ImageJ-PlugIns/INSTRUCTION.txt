1. Get ImageJ from the Website: https://imagej.nih.gov/ij/download.html
This Plug-In was developed and run on this downloaded version: bundled with 64-bit Java 1.8.0_112 (70MB)

2. Extract the downloaded ImageJ into any directory

3. Copy the contents of this folder directly into ImageJDirectory/plugins/ folder.

4. Run/Restart ImageJ
